## 🧪 Опыт работы

### Тестирование веб-приложений
- Функциональное, нефункциональное и регрессионное тестирование (фронтенд и бэкенд)
- Кросс-браузерное и кросс-платформенное тестирование
- Тестирование локализации и пользовательского интерфейса

### Тестирование API и интеграций
- Комплексное тестирование REST API (Postman, Swagger): функциональное, негативное, валидация JSON-схем
- Разработка и поддержка коллекций запросов для регрессионного тестирования
- Локализация дефектов через анализ логов (Elasticsearch) и мониторинг контейнеров (Portainer)

### Автоматизация тестирования
- Написание и поддержка автотестов на Cypress (JavaScript/TypeScript)
- Интеграция тестов в CI/CD: прохождение код-ревью, запуск в пайплайне после мержа

### Работа с инцидентами и координация
- Оперативная локализация, анализ и документирование инцидентов на продакшене
- Взаимодействие с разработчиками для устранения дефектов и предотвращения повторений
- Исполнение обязанностей QA-лида: приоритизация задач, управление нагрузкой команды, координация тестирования перед релизами

### Развитие в области ИИ
- Тестирование ИИ-компонентов и изучение применения ИИ в тестировании
- Теоретическое освоение тестирования диаризации
- Промпт-инжиниринг: генерация текстов, изображений, изучение генерации видео
- Разработка AI-агентов: реализация агента на n8n с интеграцией в Telegram, проектирование агентов в YandexGPT

### Автоматизация на Python
- Освоение автоматизации тестирования на Python: написание базовых автотестов, углубление знаний
